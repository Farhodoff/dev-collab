# Junior kontent kengaytmasi

Quyida junior bo'limi uchun ketma-ket qo'shilgan konikmalar — qisqacha tavsif, maqsadlar, amaliy mashqlar va resurslar (O‘zbekcha).

## 1. Task management (Jira / Trello)
Maqsad: Juniorga issue/ task'larni to'g'ri kuzatish va bo'laklarga bo'lish usullarini o'rgatish.

Learning objectives:
- Issue yaratishda aniq sarlavha va ta'rif yozish
- Taskni kichik qadamga bo'lish (subtasks)
- Statuslar: To Do, In Progress, Done
- Assignee va due date ishlatish

Exercises:
- Bir user story uchun 3 ta subtasks yarating va birini bajarib, statusni yangilang.

Resources:
- Atlassian Jira guide: https://www.atlassian.com/software/jira/guides

---

## 2. Communication & yordam madaniyati
Maqsad: jamoada samarali muloqot va yordam so'rash berish madaniyatini shakllantirish.

Learning objectives:
- Standupda qisqacha va aniq gapirish
- Yordam so'raganda kontekst berish (what I tried, expected, error)
- Konstruktiv feedback: kodga fokuslanish, shaxsga emas

Exercises:
- Standup uchun 3-satrlik update yozing: yesterday, today, blockers.

Resources:
- "Crucial Conversations" (kitob, umumiy maslahatlar)

---

## 3. Test yozish asoslari
Maqsad: unit test tushunchasi va oddiy test yozish qobiliyati.

Learning objectives:
- Test run qilish va test framework ishlatish (Jest yoki pytest)
- Arrange-Act-Assert pattern
- Testni izolyatsiya qilish va mock qilish asoslari

Exercises:
- Oddiy utility funksiyasi uchun 3 ta unit test yozing.

Resources:
- Jest docs: https://jestjs.io/

---

## 4. Code review basics
Maqsad: PR sharhlarini qanday samarali yozish va qabul qilish.

Learning objectives:
- PRda maqsadni aniq yozish
- Qisqa check-list: tests, lint, behavior
- Konstruktiv feedback formati: observation, impact, suggestion

Exercises:
- Misol PR oling va 3 konstruktiv sharh yozing.

Resources:
- GitHub PR best practices: https://docs.github.com/en/pull-requests

---

## 5. Vaqt boshqarish
Maqsad: vazifalarni prioritetlash va vaqtni rejalashtirish.

Learning objectives:
- Pomodoro texnikasi
- Task estimation (small/medium/large)
- Standup va weekly planningdan foydalanish

Exercises:
- 2 soatlik ish uchun Pomodoro sessiyalari bilan ish rejasini tuzing.

---

## 6. Debugging basics
Maqsad: oddiy nosozliklarni aniqlash va hal qilish usullari.

Learning objectives:
- console.log / print bilan tekshirish
- Browser DevTools: breakpoint'lar, network tab
- Loglarni o'qish va error stack trace tushunish

Exercises:
- Bir cheklangan bugni topib, breakpoint yordamida tahlil qiling va hal qiling.

---

## 7. PR yozish va amaliyot
Maqsad: samarali PR description va test qadamlarini yozish.

Learning objectives:
- PR qisqacha maqsadi va nima o'zgarganini yozish
- How to test qadamlarini qo'shish
- Related issue yoki task'ga link qo'yish

Exercises:
- Yangi PR template bo'yicha bir PR description yozing.

---
