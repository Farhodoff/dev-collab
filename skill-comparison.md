# Skill taqqoslash: Junior vs Middle vs Senior

Quyida tipik skill uchun daraja bo'yicha taqqoslash: misollar va qanday rivojlanish kerakligi ko'rsatiladi.

| Skill | Junior | Middle | Senior |
|---|---|---|---|
| Git Flow | Branch yaratish, PR | Git strategy, conflict resolution | Release strategy, branching policy |
| Code Review | Feedback qabul qilish | Feedback berish va standartlar | Qoida va jarayonlarni belgilash |
| Testing | Unit tests | Integration/E2E | Testing strategy va culture |
| Architecture | Basic API design | Service boundaries | System-wide architecture |
| Communication | Team-level | Stakeholder communication | Cross-org influence |

## Interaktiv tavsiya
- Boshlang'ich: Junior bo'limidan boshlang va asoslarni mustahkamlash. 
- Oraliq: Middledagi mentoring va code reviewga qo'shiling.
- Ustun: Senior uchun leadership va stratejiya bilan shug'ullaning.
