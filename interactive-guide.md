# Interaktiv qo'llanma va Quiz

Bu sahifa foydalanuvchilarga qaysi darajaga yaqinligini aniqlashga yordam beruvchi qisqa test va scenario asosidagi tavsiyalarni o'z ichiga oladi.

## Quiz (5 savol)
Savollar: teamwork, code review, testing, git workflow, communication.

## Scenario examples
1) PRga sharh: Reviewerdan qanday qilib address qilish kerak - step-by-step.
2) Conflict paydo bo'lganda nima qilish kerak: local rebase yoki merge?

## Tavsiyalar
- Natijaga qarab sahifada Junior/Middle/Senior uchun individual tavsiyalar ko'rsatiladi.

## Integratsiya
- app.js da mavjud quiz funksiyasi interactive qilish uchun savollarni shu faylga yoki JSON ga solamiz.
