# Middle kontent kengaytmasi

Quyida Middle darajasi uchun asosiy mavzular: code review boshqarish, mentoring, arxitektura qarorlari, CI/CD va boshqalar. Har biri uchun maqsadlar, amaliy mashqlar va resurslar.

## 1. Code Review boshqarish
Maqsad: sifatli va samarali code review jarayonini olib borish.

Learning objectives:
- Konstruktiv va maqsadga yo'naltirilgan feedback berish
- Review checklist yaratish (security, performance, tests)
- Prioritizatsiya: critical vs nice-to-have

Exercises:
- Bir PRni baholang va 5 ta aniq observation yozing: observation, impact, suggestion.

Resources:
- GitHub code review guide: https://docs.github.com/en/pull-requests/reviewing

---

## 2. Mentoring
Maqsad: Juniorlarni samarali qo'llab-quvvatlash va o'sishini rag'batlantirish.

Learning objectives:
- 1-on-1 tashkil qilish va feedback berish
- Pair programming sessiyalarni o'tkazish
- Learning roadmap tuzish va progress tracking

Exercises:
- Junior uchun 4 haftalik learning plan tuzing va baholang.

Resources:
- Effective mentoring articles (Google)

---

## 3. Arxitektura: monolith vs microservices
Maqsad: to'g'ri arxitektura tanlashni asoslarini tushunish.

Learning objectives:
- Monolith va microservices afzalliklari/cheklovlari
- Database design, API contracts, versioning
- Trade-offs va migration strategiyalari

Exercises:
- Kichik loyihani monolithdan mikroservisga aylantirish uchun qadamlar yozing.

Resources:
- Martin Fowler articles: https://martinfowler.com/

---

## 4. CI/CD Pipeline
Maqsad: avtomatlashtirilgan build/test/deploy jarayonini tashkil etish.

Learning objectives:
- GitHub Actions yoki Jenkins bilan pipeline yozish
- Rollback va environment management strategiyalari
- Test automation va gating

Exercises:
- Simple GitHub Action yozing: run tests and build on PR.

Resources:
- GitHub Actions docs: https://docs.github.com/en/actions

---

## 5. Performance va Security Best Practices
Maqsad: tizimni optimallashtirish va xavfsizlikni ta'minlash.

Learning objectives:
- Caching, indexing, profiling
- OWASP top 10, dependency scanning
- Monitoring va alerting

Exercises:
- SQL query profiling va indeks tavsiyalarini yozing; basic vulnerability checklist tuzing.

Resources:
- OWASP: https://owasp.org/

---

## 6. Technical leadership
Maqsad: texnik qarorlarni jamoa bilan muvofiqlashtirish va liderlik qilish.

Learning objectives:
- Decision-making framework
- Tech roadmap va tech debt management
- Cross-team communication

Exercises:
- Bir feature uchun tech spec yozing: options, pros/cons, recommended approach.
